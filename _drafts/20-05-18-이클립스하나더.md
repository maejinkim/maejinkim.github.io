mac에서 이클립스를 두개 키고 싶다.

하나의 이클립스가 실행되어 있다면, 
터미널을 키고 Eclipse.app이 설치되어있는 폴더로 이동한다.

주로 루트안의 eclicpse나, applications안에 설치가 되어있다.

나같은 경우는 ~/eclipse/java-2019-09/에 설치되어 있었다.

``` bash
cd ~/eclipse/java-2019-09/
```

이동 한 후에 아래 명령어를 입력하여 이클립스를 실행하면 된다. 

``` bash
open -n Eclipse.app
```
